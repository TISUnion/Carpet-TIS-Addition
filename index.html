Rich Features & Tools

Carpet TIS Addition packs an array of features and tools, including carpet rules, loggers and commands, enriching your Technical Minecraft experience

Keep the game vanilla

Carpet TIS Addition never changes the vanilla behavior unless explicitly configured by the user. Keep Minecraft vanilla

Extensive Version Support

From Minecraft 1.14 to the latest snapshot, Carpet TIS Addition provides consistent support across multiple Minecraft versions
