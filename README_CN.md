这是一个 [Carpet mod](https://github.com/gnembon/fabric-carpet) (fabric-carpet) 的扩展 mod，包含了不少~~NotVanilla的~~有意思的功能以及特性

在默认配置条件下，本模组不会改变任何原版机制

跟同 Minecraft 版本的 carpet mod 一起使用即可。尽可能地使用较新的 carpet mod
